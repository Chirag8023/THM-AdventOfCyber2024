Just a memoir of the challenges

Day Challenge
1.

OPSEC ( Purple Teaming )

We explored a YouTube-to-MP3 website which was installing malware to its users and analyzed the meta-data of the malware through which we backtracked to the attacker's actual identity.
  • Commands & Tools Used: Find, Exiftool, Github
2.

Log Analysis ( Blue Teaming )

We investigated security logs of a website, found patterns in the authentication, detected a brute force attack, and examined the encoded PowerShell command that had been run on the systems.
  • Commands & Tools Used: Elastic SIEM, CyberChef
3.

Log Analysis ( Purple Teaming )

This time, we used ELK to search through logs and found a specific IP that was using a web shell through a profile picture upload. In the second half, we logged into a website as admin because of weak/common credentials and tried to exploit an RCE via file upload.
  • Commands & Tools Used: ELK (Elasticsearch, Logstash, and Kibana), ls, cat
  • Techniques Used: RCE (Remote Code Execution)
4.

Atomic Red Team ( Purple Teaming )

We recreated MITRE ATT&CK technique `T1566.001 Spearphishing` using the Atomic Red Team library, then in Windows Event Logs analyzed Sysmon logs confirming the compromise. Next, we tested technique `T1059.003 Command and Scripting Interpreter: Windows Command Shell`. After iterating through multiple tests, we found the one from which the machine was vulnerable.
  • Commands & Tools Used: Atomic Red Team Library, Sysmon, Windows Event Logs
5.

XXE ( Red Teaming )

We used `Burp Suite` to intercept requests and responses and tried to find out if the website is vulnerable to XXE (XML External Entity) Injection, then exploited it to get data only admins can access.
  • Commands & Tools Used: Burp Suite
  • Techniques Used: XXE (XML External Entity) Injection
6.

Sandboxes ( Blue Teaming )

We tested a malware in a virtual machine and detected it with `YARA rules`. After getting detected, we obscured it by using encoded commands, but it was still detectable by `floss` and `sysmon logs`.
  • Commands & Tools Used: YARA, Sysmon, CyberChef, FLOSS
7.

AWS Log Analysis ( Blue Teaming )

We went through Amazon CloudTrail and RDS logs using `jq` and `grep` to find out who was actually responsible for altering the QR code on the donation website.
  • Commands & Tools Used: JQ, grep
8.

Shellcodes ( Purple Teaming )

Using `msfvenom`, we generated a shellcode, then tried to achieve a reverse shell by executing it on the target machine by loading it into memory.
  • Commands & Tools Used: msfvenom, nc
9.

GRC ( Blue Teaming )

We performed a Risk Assessment based on the answers provided by different vendors to choose the one with the lowest risk.
10.

Phishing ( Red Teaming )

We used `Metasploit Framework` to create a malicious macro and sent it through a similar-looking email. When the macro file was opened, we got access to the target system.
  • Commands & Tools Used: Metasploit Framework
  • Techniques Used: Typosquatting
