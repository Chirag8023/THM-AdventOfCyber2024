Advent of Cyber 2024 Challenges
Day Challenge
1.

OPSEC (Purple Teaming)

We explored a YouTube-to-MP3 website which was installing malware to its users and analyzed the meta-data of the malware through that we backtrace to the attacker's actual identity.
Commands & Tools Used: Find, Exiftool, Github.
2.

Log Analysis (Blue Teaming)

We Investigated security logs of a website and found patterns in the authentication and detected a brute force attack, then examined the encoded powershell command that had been run on the systems.
Commands & Tools Used: Elastic SIEM, CyberChef.
3.

Log Analysis (Purple Teaming)

This time, we used ELK to search through logs and found a specific IP that was using a web shell through profile picture upload. In the second half, we logged into a website as admin due to weak/common credentials and tried to exploit an RCE via file upload.
Commands & Tools Used: ELK (Elasticsearch, Logstash, and Kibana), ls, cat.
Techniques Used: RCE (Remote Code Execution).
4.

Atomic Red Team (Purple Teaming)

We recreated MITRE ATT&CK technique `T1566.001 Spearphishing` using the Atomic Red Team library, then in Windows Event Logs analyzed Sysmon logs confirming the compromise.
Commands & Tools Used: Atomic Red Team Library, Sysmon, Windows Event Logs.
5.

XXE (Red Teaming)

We used Burp Suite to intercept request and response and tried to find if the website is vulnerable to XXE (XML external entity) Injection, then exploited it to get data only admins can access.
Commands & Tools Used: Burp Suite.
Techniques Used: XXE (XML External Entity Injection).
6.

Sandboxes (Blue Teaming)

We tested a malware in a virtual machine and detected it with YARA rules. After getting detected, we obscured it by using encoded commands, but it was still detectable by FLOSS and sysmon logs.
Commands & Tools Used: YARA, Sysmon, CyberChef, FLOSS.
7.

AWS Log Analysis (Blue Teaming)

We went through Amazon CloudTrail and RDS logs using jq and grep to find out who was responsible for altering the QR code on the donation website.
Commands & Tools Used: jq, grep.
8.

Shellcodes (Purple Teaming)

Using msfvenom, we generated a shellcode and tried to achieve a reverse shell by executing that on the target machine by loading it to memory.
Commands & Tools Used: msfvenom, nc.
9.

GRC (Blue Teaming)

We performed a Risk Assessment based on the questions answered by different vendors to choose the one with the lowest risk.
10.

Phishing (Red Teaming)

We used Metasploit Framework to create a malicious macro and sent it through a similar-looking email. When the macro file opened, we gained access to the target system.
Commands & Tools Used: Metasploit Framework.
Techniques Used: Typosquatting.
11.

Wi-Fi Attacks (Red Teaming)

We set our Wi-Fi into monitor mode using iw, then used airodump-ng to scan for nearby networks, sent deauth packets to a specific BSSID, captured the 4-way handshake, and performed a dictionary attack using aircrack-ng.
Commands & Tools Used: iw, airodump-ng, aircrack-ng, aireplay-ng.
